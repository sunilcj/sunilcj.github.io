# Hey, I'm Sunil CJ 👋

### What I Do

I lead the design and development of **microservice architectures at scale** — spanning authentication, billing, real-time search, data pipelines, and third-party integrations across two major products.

**FlexWhere** — Workplace management platform for hybrid offices
- Core services: auth, search engine, billing, tray/desktop clients, import/export, Google Calendar sync
- Activity manager with organization insights and performance dashboards

**EdControls** — Construction quality control & snag management platform
- API services, project lifecycle management (S3 Glacier archival/restore), bulk operations
- Billing, analytics, data hub services, and indoor path tracking (IMU-based)
